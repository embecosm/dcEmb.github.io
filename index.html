dcEmb¶

This is the documentation for the dcEmb repository, Embecosm’s high performance implementation of Dynamic Causal Modelling in C++.

Early Release Warning¶

This project is still under active development, and will undergo significant changes. Caveat Empor!

Getting Started¶

See our installation instructions for instructions on installing the software on your platform. See current models for details on existing DCM models. See our building your own model for instructions and a worked example of how to implement your own model.
